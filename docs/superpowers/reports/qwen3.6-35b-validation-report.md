### 5.3 发现的问题与建议

#### 问题 1: 因果推理 HTTP 400 失败
- **现象**: reasoning-distilled 模型输出 thinking block 中包含 `\x1a` EOF 字符
- **影响**: 因果推理子类别完全失败 (0/25)
- **建议**: 已添加 try-except 降级保护，建议 LM Studio 端修复 input 解析逻辑

#### 问题 2: 部分子类别区分度不足
- **现象**: 逻辑推理 3 个测试全部满分 (100%)
- **影响**: 无法区分模型上限
- **建议**: 增加更高难度的逻辑谜题 (如模态逻辑、时态逻辑)

#### 问题 3: API 接口开发评分偏低
- **现象**: FastAPI CRUD 接口仅 3/30 (10%)
- **分析**: 可能是 prompt 不够清晰，或评分标准过于严格
- **建议**: 审查评分规则，确保评分能反映模型真实能力

#### 问题 4: 多轮对话评估有 bug
- **现象**: 上下文理解子测试报错 `'content'` KeyError
- **影响**: 多轮对话得分 9/35 (25.7%)，可能被低估
- **建议**: 修复评估代码中的 KeyError
